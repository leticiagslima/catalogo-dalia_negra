A pasta Dados engolba o arquivo: dados-produtos.jsx. 
Esse arquivo contém as informações que serão utilizadas pelos cartões e pelo perfil, e serão apresentadas no site nos cartões dos produtos.

Guarda as seguintes informações:
1. Imagens dos produtos;
2. Nome;
3. Preço,
4. Categoria (essas farão parte da interface do site, logo, o usuário poderá navegar pelo site com mais facilidade, filtrando os produtos que gostaria de ver pelo tema).
