A pasta Componentes engolba os arquivos: cartoes.jsx; produto-perfil.jsx; slideshow.jsx e topo.jsx. 
Eles são os arquivos que primeiro serão visualizados pelo usuário, e são o "centro" do site.

1. Cartões: contém a estruura dos cartões que serão visualizados com os produtos no site. (Utiliza props)
2. Produto-Perfil: é utilizado como import no arquivo de cartões, e serve como base para a estrutura dos mesmos. (Utiliza props)
3. Slideshow: as imagens que serão apresentadas na página incial do site, apresentando os produtos e a loja. (Utiliza useEffect e const para fazer a mudança automática dos slides)
4. Topo: este arquivo contém as informações de nome apresentadas no topo da página, como o nome da loja.
