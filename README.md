# catalogo-dalia_negra
Catálogo digital da loja fictícia Dália Negra, desenvolvido com React na UC Desenvolvimento de Sistemas.

Esse projeto de site é para uma loja que vende itens de colecionador, inspirados em filmes, séries livros, como: O Senhor dos Anéis, Harry Potter, Star Wars e Jogos Vorazes.

Para a criação deste catálogo foi utilizado: Framework React, CSS e Linguagem JavaScript, através do Vite no Visaul Studio Code.

Este católogo possui uma estrutura organaizada em pastas de arquivos, que compoem o site e tudo o que o usuário vê e interage. 
Essas pastas e suas funcionalidades podem ser vistas e explicadas nos arquivos de READ ME, espalhadas pelo catálogo do site.
